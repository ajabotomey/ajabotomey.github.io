Hacky RPG (tentative name)

This is a small project that I have been working on since the start of my 4th year of university. The game is currently developed in C# with the MonoGame Framework and is currently targeting Windows (XP, Vista and 7), Mac OS X, Linux and PlayStation Mobile (PS Vita only). Once these versions are complete, a mobile version may be available as well as Windows 8 support.

So far, I have implemented a tile engine with collision detection. The level and objects are loaded through an XML file which have links to the relevant resources.

You can access the GitHub repository from here. Only the Windows version is available on GitHub at the moment although that code also works within PlayStation Mobile Studio.
