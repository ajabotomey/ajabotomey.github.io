OpenGL Game Engine


Overview

This particular project was the second assignment of the course 3407ICT - Graphics Programming. This game engine utilises OpenGL to draw objects to the screen using Vertex Buffer Objects.


Progress

This engine sucessfully reads OBJ files and draws those objects using OpenGL Vertex Buffer Objects. Also available

  • The generation of terrain using a heightmap.
  • Textured objects.
  • Phong shaders.
  • Bump/Normal mapping.
  • The use of a framebuffer object to render shadows.
  • Camera transformations from a perspective view.
