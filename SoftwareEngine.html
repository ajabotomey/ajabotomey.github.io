Software Rendering Engine


Overview

This particular project was the first assignment of the course 3407ICT - Graphics Programming. This game engine utilises Software Rendering to draw all the graphics to the screen.


Progress

This engine sucessfully renders a 3D object using Software Rendering techniques. This engine is also capable of drawing 3D objects loaded from file as well as the ability to rotate, translate and scale those objects. The program also utilizes Liang-Barsky line clipping so the objects continue to fly off the screen.
